# COMMAND

This isn't a file, but data specification that appears instantiated in memory.

## COMMAND STRUCTURE

| Offset | Type  | Description
|---------|--------|------------
| 0x0     | uint16 | Command ID
| 0x2     | uint8 | Level
| 0x3     | uint8 | Flag
| 0x4     | uint16 | Experience
| 0x6     | uint16 | Ability ID
